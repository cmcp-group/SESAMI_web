Tired of using an old excel spreadsheet from your co-workers?

On-the-fly, this website performs the Brauner-Emmett-Teller (BET), Excess Sorption Work (ESW), BET-ESW, and Machine Learning (ML) analysis to accurately estimate the surface area in nanoporous materials. You can upload your data by using either Adsorption Information File (AIF) or Comma Separated-Values (CSV) file format.

Step 1: Convert RAW output files from commercial equipments to AIF format using this link.

Step 2: Upload the converted data from Step 1 by clicking "Upload CSV or AIF".

Step 3: Click "Run Calculation"

Default gas molecule for the analysis is Nitrogen. If you are running calculation with the "sample data", make sure to click Show general options below to change the gas molecule to Argon! Remember p0 = 1e5 Pa

Calculations assume the input data file contains isotherm measured at one of the following conditions:
• Nitrogen: T = 77 K
• Argon: T = 87 K

If you want to analyze BET isotherm from simulation, download the example CSV file and format your simulation data accordingly.
